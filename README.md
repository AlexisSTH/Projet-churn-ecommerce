# Prédiction de Churn Client E-commerce | Olist

Projet de Machine Learning pour identifier les clients à risque de churn 
sur la plateforme e-commerce brésilienne Olist.

## Objectif business
Permettre à l'équipe CRM de cibler les clients à risque pour des campagnes 
de rétention, avec un ROI mesurable.

## Stack technique
- **SQL** (DuckDB) : extraction et agrégations
- **Python** : feature engineering, modélisation (scikit-learn, XGBoost)
- **Power BI** : dashboard décisionnel

## Pipeline
1. Exploration SQL et data quality
2. Feature engineering (RFM + comportemental)
3. Modélisation (Random Forest / XGBoost)
4. Dashboard Power BI
